To install:
------------

1.  At command prompt, cd to this directory
2.  pip install .

To check installation:
----------------------

Run python, type 'import jacks' and check that no error is thrown.


To run JACKS on screen data:
----------------------------

python run_JACKS.py countfile replicatefile:replicate_hdr:sample_hdr:ctrl_sample sgrnamappingfile:sgrna_hdr:gene_hdr outprefix

see 2018_paper_materials/README.txt for examples.
